Origin of Vue

Name: Evan You

former Google developer where he worked with AngularJS on a number of porjects. After some time he thought to himself:

"What if I could just extract the part that I really liked about Angular and build something really lightweight."

-> first source code commit to the project was dated in July 2013

--> the project was then released in February 2014 with version 0.9 Animatrix

Frameworks vs. Programming Languages

Programming Languages are USED to create Frameworks

Frameworks are targeted to help solve specific usecases and help developers working faster and more efficient

as an example:

C# -> ASP.NET MVC -> Wep Apps

Frameworks vs. Libraries

"Your code calls the library, but the framework calls your code"

or to put it in other words:

a Framework acts as your foundation and a library solves specific problems
